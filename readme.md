# Battleships

## Prerequisites
* .NET 6

## Building application
Run
```
dotnet build
```

## Running application:
Through these ways: 
* powershell script:
```
./run.ps1
```
* dotnet command
```
dotnet run --project ./battleships.main/battleships.main.csproj
```
* building and finding executable in the bin folder of the battleships.main project
* vscode

## Testing
```
dotnet test
```
